📌 데이터 출처

국토교통부 아파트 매매 실거래가 상세 자료 API (공공데이터포털)
실제 거래가 신고된 계약 건만 포함되며, 추정·호가 데이터는 제외됩니다.

🔍 필터 기준

  • 지역: 서울시 25개 구 전체
  • 거래금액: 25억 원 이하 (예산 상한선)
  • 전용면적: 84㎡ 이상 (4인 가족 최소 기준, 약 30평대)
  • 기간: 최근 12개월 실거래 데이터

🏞️ 한강 인근 기준

마포구·용산구·영등포구·성동구·광진구·동작구·강동구·강남구·서초구·송파구
한강 접근성과 조망권이 상대적으로 우수한 구를 분류한 것으로, 실제 단지별 조망 여부와는 다를 수 있습니다.

📈 통계 계산 방식

  • 평균 거래가: 필터 조건 내 전체 거래의 산술 평균
  • 거래량 추이: 월별 신고 건수 집계
  • 면적대 분포: 84~95 / 95~115 / 115~135 / 135㎡ 이상으로 구분

🔄 업데이트 주기

GitHub Actions를 통해 매일 오전 9시(KST) 자동 갱신됩니다.
국토부 데이터는 거래 신고 후 30일 이내 등록 의무가 있어 최신 거래는 1~4주 지연될 수 있습니다.
